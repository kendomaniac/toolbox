# Append a timed directory with the format %Y%m%d\_%H%M%S to the target_directory path.

Append a timed directory with the format %Y%m%d\_%H%M%S to the
target_directory path.

## Usage

``` r
modifier_timed_directory(target_directory)
```

## Arguments

- target_directory:

  The target directory.

## Value

Timed directory with the format %Y%m%d\_%H%M%S appended to
target_directory path.
